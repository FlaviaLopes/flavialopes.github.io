Mais que dados – esta é a minha história

Eu sou uma estrategista de dados

Nem todo mundo começou em ciência de dados já sabendo que queria estar aqui. No meu caso, tudo começou em uma aula de banco de dados, onde trabalhei com um dataset sobre eleições federais, normalizando tabelas e estruturando consultas. Foi ali, ao cruzar variáveis aparentemente isoladas e revelar padrões ocultos, que percebi algo fascinante: muitos dados valiosos estão enterrados em estruturas complexas, esperando para serem descobertos.

Desde então, me apaixonei pelo desafio de tornar os dados mais acessíveis e compreensíveis. Ao longo da minha trajetória, participei de projetos de análise de dados, muitos deles de ponta a ponta, sempre buscando transformar informações brutas em insights estratégicos que realmente fazem a diferença na tomada de decisões.

Ciência de Dados por analogia

Eu investigo a cena do crime

A ciência de dados é como uma investigação criminal. Os dados estão espalhados como pistas em uma cena do crime—alguns são relevantes, outros são ruídos.

Como uma investigadora, eu:

  • 1. Coleto evidências – busco todas as fontes possíveis (bancos de dados, APIs, planilhas)
  • 2. Analiso pistas – faço a triagem do que realmente importa, limpando e organizando os dados
  • 3. Construo e valido hipóteses – uso a estatística e modelos para entender o que aconteceu
  • 4. Encontro a prova definitiva – descubro e reúno os padrões ocultos que explicam o que houve
  • 5. Conto a história do crime – apresento as evidências de forma clara para resolver o caso e convencer o júri (stakeholders).

Eu extraio o ouro da mina

Os dados brutos são como ouro incrustado nas rochas. Você pode estar ao lado de uma mina que vale milhões, mas nem sempre sabe disso. E, mesmo que saiba, talvez não tenha os recursos para extrair esse tesouro.

Como uma mineradora de dados, eu:

  • 1. Faço a prospecção – exploro terrenos (bases de dados, APIs, logs) para identificar onde há potencial
  • 2. Extraio os filões – coleto os dados brutos, mesmo aqueles desorganizados e de difícil acesso
  • 3. Processo o material – limpo, transformo e organizo os dados para separar o que realmente importa, o seu ouro
  • 4. Analiso a composição – uso estatística e machine learning para identificar padrões valiosos, e avaliar o quão valioso é o seu material
  • 5. Entrego o ouro 24K – refino o material entregando algo valioso e aplicável para a tomada de decisão - a informação

No final, assim como um perito explica a cena do crime, e o minerador busca o ouro dos profundos filões, um cientista de dados recupera e explora os dados brutos para transformá-los em conhecimento estratégico.
